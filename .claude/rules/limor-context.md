# Limor Sub-Project Context Rule

**Scope**: All tasks related to Limor demo videos
**Auto-Load**: `Limor/CONTEXT.md`

---

## When to Load Limor Context

ALWAYS read `Limor/CONTEXT.md` when working on:
- Video compositions (MobileChatDemoV2, MobileChatDemoV3)
- Screenshot capture scripts
- Scene components
- Dark mode CSS
- Touch animations
- Any Limor-related assets

## Quick Reference

```bash
# Load context first
Read Limor/CONTEXT.md

# Then proceed with task
```
